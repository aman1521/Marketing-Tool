# CRO Analysis

## Steps

1. Identify friction points in user flow.
2. Evaluate headline clarity and value proposition.
3. Analyze CTA placement, size, and messaging.
4. Check form complexity and perceived effort.
5. Measure page load performance and visual weight.

## Framework: The LIFT Model

- **Value Proposition**: The core driver of conversion. Must be clear.
- **Relevance**: Does the page match the expectation of the source link?
- **Clarity**: Is it obvious what the page is about?
- **Urgency**: Why should the user act right now?
- **Anxiety**: What objections or fears does the user have?
- **Distraction**: Are there conflicting goals/links on the page?

## Output
